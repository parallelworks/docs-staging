View/Edit input and output configuration

Type over green text to edit input format, type, and label. See form preview adjust to type and label changes to confirm edits are successful.

Input For each input of your workflow, you will need to to pick a type and label (format and help are optional). The input specifications define how the input form will be rendered and appear to workflow users.

Label: Indicate the desired name of your input.

Format: If a file type is specified, indicate file type (Tip: Do not include the ‚Äú.‚Äù in your format).

Help: Add text below input with helpful notes and direction.

Type: Indicate format of input.

Types Supported

Data:

Format: Extension of input file type, do not include ‚Äú.‚Äù in format type.

Text:

Value: you can enter default value.

Integer: Will render as a slider accepting only intergers. Values: specified minimum and maximum values.

Float: Will render as slider accepting all numbers. Values: specified minimum and maximum values.

OUTPUT All workflows will always return Type: Data. Users will need to specify both format and label for identification of outputs in your workspace upon completion of your workflow. Hint: Output format and label are what will appear as your results output in the workspace upon completion of your workflow runs.
