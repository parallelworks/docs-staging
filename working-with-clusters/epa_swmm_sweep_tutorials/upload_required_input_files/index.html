Upload Required Input Files

The SWMM parametric sweep requires TWO key input files\ (1) A SWMM Templated Input file(.template) (2) A SweepFile (.sweep)

Select the Upload icon in the upper right corner of the Tools & Workflows panel to upload the required input files.

Once upload is complete, click CLOSE in the lower right corner of the upload status screen
