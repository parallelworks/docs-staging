Workspace: Run Outputs

Upon completion of your workflow run, the results files will display in green in your currently active workspace. View data and details, edit, annotate or download these output files.

View Data See up to the first MB of data in the file.

\ \

Edit Content & Attributes Access File Attribute Editor. Update attributes, data type and permissions. Click file name to expand and access more options.

More Information View file details (date created, size, format, API ID etc.).

\ \

Save Download file to your hard drive.

\ \

Annotate Add tags and notes.
