View and Edit Resources

Click the resource to view the main edit interface.

Settings Edit the resource description and edit privacy settings in the settings panel on the main page.

Definition On the definition tab, configure your resources using the drop down menus.

Resource Type At present, AWS EC2 resources are the default compute resources; however, Google Cloud Platform GCE is also available. Larger HPC resources with InfiniBand interconnect can be added with custom compute configurations.

Instance Select desired instance type. You can create a new resource for each workflow you run, or edit your default resource configuration to suit your workflow needs.

AMI An Amazon Machine Image (AMI) will be assigned by default. If you have your own AMI configuration, you can overwrite the AMI number provided.

Workers/Instance Select desired number of workers you would like to have running per instance based on workflow needs.

Max Workers Select maximum number of workers you would like to have active.

Hint: In order to most effectively reconfigure your resources and execute your workflow, turn off any resources that are changing before making any edits. After completing edits, turn resources back on and the resource updates should take effect.
